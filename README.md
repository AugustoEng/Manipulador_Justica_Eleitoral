# Manipulador_Justica_Eleitoral
Trabalho bimestral da matéria Estrutura de Dados ministrada pelo professor Marcelo Eustaquio do curso Engenharia de Software da Universidade Católica de Brasília. O trabalho por sua vez exige um programa que leia, concatene e manipule os dados de diversos arquivos CSV, além de possuir uma interface de pesquisa por municípios, realizando um SELECT
